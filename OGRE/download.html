Download

I have packaged OGRE and its dependencies into a self-contained executable for Mac OS X. Get it here:

OGRE for Mac.

If you'd like to play around with it, download the sample output file at the bottom of this page, and check out the User's Guide! I am currently working on a Linux version. Whatever your operating system, since the code is written with Qt (which is designed to be multi-platform), you can always download the source code and compile it on your own machine. If you have the version control software git, you can get the latest version by typing the following on the command line:

git clone https://github.com/dtamayo/OGRE.git

If you don't, you can go here:

https://github.com/dtamayo/OGRE

and download a zipped folder with the code (on the bottom right). You will also need a couple dependencies--e-mail me at if you'd like to do this and I'll help you sort them out (I am in the process of writing a how-to guide for getting them). Getting the source code is great because then you can edit it for your own purposes, using the extensive Source Code Documentation, which includes sections like 'How do I add a menu item?' etc. (go to the related pages tab).

Currently the code can only read SWIFT files. If you use a different integrator, let me know and I can add that format! I still need to write a routine that can read SWIFT bin.dat files (let me know if you could help with this!). For now, I have written a simple bash script that will repeatedly call tool_follow and concatenate all the particle histories into one text file. This is the format that OGRE can read. Download the script below:

gen_swift.zip,

put it in the folder where you executed your run (where bin.dat and the input and parameter files reside), and edit the first couple lines (just asks for simple things like the name of the parameter file). Then run it.

Here's a sample .swift file, so you can quickly play around with OGRE. Make sure to check out the User's Guide!

mearth.swift.zip.
