Installing from Source (Linux)

  • Get Qt

    • OGRE uses the amazing user-interface framework Qt for all its windows, buttons, menus etc. To compile the source code you'll need this first. Get it at qt-project.org/downloads.

  • Get Git

    • Git will allow you to pull the latest version of the OGRE source code (and lots of other programs!), and update to newer versions easily. It will also allow you to contribute to the code! Follow the simple instructions here: git-scm.com/book/en/Getting-Started-Installing-Git Special note for Fedora/Red Hat Linux users: I'm running Red Hat Enterprise Linux Server release 5.9 (check by running cat /etc/issue on the command line). When I tried 'yum install git' I got 'No package git available'. I had to install EPEL, a list of extra Fedora packages for yum to recognize, which includes git. Get the latest version specific to your OS version at https://fedoraproject.org/wiki/EPEL.
